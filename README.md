# Village Defender

Village Defender is a 2D survival defense game built with PixiJS, TypeScript, and Vite.

This repository is structured as a professional portfolio project for a Game Web Intern interview. The implementation emphasizes clean architecture, maintainability, and deployment readiness.

## Status

The project is being built incrementally through focused milestones.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
